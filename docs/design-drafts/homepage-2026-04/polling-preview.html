Methodology · open-source & auditable

How we average polls.

We follow RealClearPolitics' aggregation method: trailing public polls, normalized by methodology and recency. The polling pipeline runs nightly. Campaign-finance data is mirrored from CAL-ACCESS continuously.

RealClearPolitics· Wikipedia poll aggregator· CAL-ACCESS· FEC (federal-side IE)
  1. 01 POLLS COLLECTED Public horse-race polls from RCP and Wikipedia, normalized to a common schema
  2. 02 DEDUPED & STAGED Aggregator rows preferred; raw polls merged when no aggregator value exists
  3. 03 FINANCE MIRRORED CAL-ACCESS RCPT_CD, EXP_CD, IE filings synced; donors normalized for clean rollups
  4. 04 ROLLUPS COMPUTED Per-candidate raised, spent, cash, IE for/against, top donors, top industries
  5. 05 PUBLISHED Polling avg + finance rollups served via API and UI; sources cited per row
